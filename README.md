# 🛒 Comprar

O **Comprar** é um aplicativo mobile desenvolvido durante o curso de Fundamentos de React Native. Ele permite criar e gerenciar listas de compras de forma simples e prática, ajudando o usuário a organizar seus itens do dia a dia.

## 📱 Funcionalidades

- Adicionar itens à lista
- Remover itens
- Visualizar lista em tempo real
- Interface simples e intuitiva

## 🚀 Tecnologias

- React Native
- Expo
- JavaScript / TypeScript

## ⚙️ Como rodar o projeto

### Pré-requisitos

- Node.js instalado
- Expo CLI (opcional, pode usar via npx)
- Um emulador Android/iOS ou o app Expo Go no celular

### Passos

1. Clone o repositório:

```bash
git clone https://github.com/NattanSilva/comprar-app.git
```

2. Acesse a pasta do projeto:

```bash
cd comprar-app
```

3. Instale as dependências:

```bash
npm install
```

4. Inicie o projeto:

```bash
npx expo start
```

5. Abra no dispositivo:

- Use um emulador, ou
- Escaneie o QR Code com o **Expo Go**

## 📌 Observações

Este projeto foi desenvolvido com foco educacional para praticar conceitos básicos de React Native como componentes, estado e estilização.
